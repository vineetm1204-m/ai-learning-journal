# Day 28 — The Transformer architecture (Attention is All You Need)

## 🧠 Concept Explanation

The Transformer replaces recurrence and convolution with a pure attention mechanism, allowing every token in a sequence to attend to all others in parallel. At its core are multi‑head self‑attention layers that compute weighted sums of value vectors, where the weights come from a scaled dot‑product between query and key vectors. By projecting the same input into multiple heads, the model captures diverse relational patterns (e.g., syntactic, semantic) simultaneously. Positional encodings inject order information because attention itself is permutation‑invariant. Each encoder block stacks a self‑attention sub‑layer, a feed‑forward network, residual connections, and layer normalization. The decoder mirrors the encoder but adds a cross‑attention sub‑layer that attends to the encoder’s output, enabling sequence‑to‑sequence tasks like translation. Training uses teacher forcing with a masked self‑attention mask to prevent future tokens from influencing predictions. The architecture’s parallelism yields massive speed‑ups on GPUs/TPUs and scales to billions of parameters, powering models such as BERT, GPT, and T5.

---

## ✅ Key Points

- Self‑attention computes context for each token by weighting all tokens in parallel.
- Multi‑head attention learns multiple representation subspaces simultaneously.
- Positional encodings provide sequence order without recurrence.
- Encoder‑decoder stacks with residual connections and layer norm enable deep training.
- Masked self‑attention in the decoder enforces autoregressive generation.

## 💡 Real-World Analogy

Imagine a round‑table discussion where every participant instantly hears everyone else’s comments and can weigh each contribution according to relevance, rather than waiting for a single speaker to finish.

---

## ⚠️ Common Mistakes

- Forgetting to add positional encodings, causing the model to treat sequences as bags of tokens.
- Using the same mask for encoder and decoder self‑attention, leaking future information during training.
- Neglecting residual connections and layer normalization, leading to vanishing gradients in deep stacks.
